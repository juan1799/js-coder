Simulador interactivo de gestión de alumnos

Simulación de sistema de gestión de alumnos y notas

La simulación comienza cuando el usuario ingresa que desea simular el sistema de gestión de alumnos y notas

Luego el usuario deberá ingresar creará un nuevo usuario con su nombre, su apellido y la contraseña

El sistema solicitará el apellido y la contraseña del profesor, una vez completo el ingreso permitirá al usuario cargar nuevos alumnos

Instructivo

  • 1- Seleccionar SI para seguir con la simulación
  • 2- Crear un usuario nuevo ingresando tu nombre y contraseña
  • 3- Ingresar al sistema con el usuario anteriormente creado
  • 4- Cargar un nuevo alumno
